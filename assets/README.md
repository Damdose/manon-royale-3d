# Dossier d'assets — Manon Royale 3D

Dépose ici tes images **PNG** (avec fond transparent pour les personnages/tours).
Le jeu les charge automatiquement au démarrage. Si un fichier manque, le jeu utilise une forme 3D de secours (donc tu peux en ajouter une par une).

➡️ Après avoir ajouté/modifié des images : recharge la page avec **Cmd + Shift + R**.

---

## 1) Troupes  (PNG transparent, personnage **vu de face**, **pieds en bas du cadre**, carré ~512×512)
Nomme chaque fichier `troop_<clé>.png` :

- `troop_manon.png`      (Manon — fille blonde en rose)
- `troop_caniche.png`    (caniche blanc)
- `troop_sac.png`        (sac à main doré)
- `troop_pigeon.png`     (pigeon volant)
- `troop_meme.png`       (mémé à lunettes)
- `troop_diva.png`       (diva)
- `troop_mannequin.png`  (mannequin de couture bleu)
- `troop_talon.png`      (escarpin rouge)
- `troop_damien.png`     (homme en noir)
- `troop_vespa.png`      (scooter)
- `troop_valise.png`     (valise)
- `troop_pump.png`       (bâtiment "Boutique")
- `troop_tourelle.png`   (tourelle/canon)

## 2) Tours  (PNG transparent, **vue de face**, **base en bas du cadre**, ~512×640)

## 3) Terrain (optionnel)
- `arena.png`  — texture du sol vue de dessus (gazon + lanes + rivière). Ratio **480 × 620** (ex. 960×1240). Remplace le sol dessiné par code.

## 5) Fond (optionnel)

---

### Comment générer ces images ?
- Un outil d'IA d'image (DALL·E, Midjourney, etc.) : prompt type « cute 3D cartoon character, front view, full body, transparent background, mobile game style ».
- Ou des packs d'assets libres (itch.io, kenney.nl…).
Évite de copier les vrais assets de Clash Royale (droits d'auteur).
